# Autism Screening using Machine Learning

## Objective
To predict Autism Spectrum Disorder (ASD) using machine learning techniques.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix

## Result
Random Forest performed best based on F1 Score.

## Tools Used
- Python
- Google Colab
- Scikit-learn
